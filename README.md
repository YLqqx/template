# 前端快速搭建框架

vue 快速搭建

## 安装

```shell
$ git clone https://github.com/Qiu-zm/template.git

$ npm install
```

## 运行

```shell
$ npm run dev
```

或者

```shell
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

## vue2

vue2 基础包含 i18n vant router vuex

vue2-web3 增加web3.js

vue2-web3-axios 增加 axios

## vue3

vite-vue3 基础包含 vant vuex axios
